Provides support for the 

artifact
URL protocol

The 

artifact
URL protocol provides a way to resolve an artifact, and its (transitive) dependencies.

The Handler class defines a stream handler for artifact URLs. You can specify that this class be used for artifact URLs by using the following code:

URL.setURLStreamHandlerFactory(new ArtifactURLStreamHandlerFactory()); or must be set via a JVM property such as: -Djava.protocol.handler.pkgs=org.rioproject.url
