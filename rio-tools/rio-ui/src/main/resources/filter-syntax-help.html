Filter query syntax is simple. You have operands and keywords

Keywords
type The event type. The type is the first column. Examples here are OPSTRING_DEPLOYED or PROVISION_FAILURE
desc The event description. The description is the second column.

Operands
= The equal sign, indicating you want an exact match
is Synonomous with =
contains Indicates that the filter should match an event entry if the type of desc contains the text
~ Synonymous with contains

Examples:
type ~ OP The filter will match on all event type that contains the string OP
type = BREACHED desc ~ Hello
 The filter will match on all event types equal to BREACHED and that have Hello in the description
